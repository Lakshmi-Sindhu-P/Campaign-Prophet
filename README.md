# Marketing-Campaign-ROI-Prediction
This project analyzes financial marketing campaign data to measure campaign performance, calculate ROI, validate business assumptions using statistical hypothesis testing, and predict customer subscription likelihood using logistic regression.

The project uses Python for data analysis and modeling, SQL/PySpark for scalable campaign reporting, and Hive-style SQL logic to simulate enterprise analytics workflows. The final output includes customer segmentation, ROI optimization, and actionable recommendations for marketing acquisition strategy.
